# Personal Website

A simple static site — no build tools needed.

## Structure

- `index.html` — About / home page
- `blog.html` — Writing index
- `posts/` — individual blog posts
- `contact.html` — Contact page
- `css/style.css` — all styling (colors are CSS variables at the top)

## Preview locally

Open `index.html` in a browser.

## Add a blog post

1. Copy `posts/first-post.html`, rename it
2. Edit the title, date, and content
3. Add a link to it in `blog.html`

## Deploy

Works as-is on GitHub Pages, Netlify, or Cloudflare Pages.
